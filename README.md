# Evaluating Sequence Models on Word Problems and Related Prediction Tasks

This is the official code repository for the paper: [Why Depth Matters in Parallelizable Sequence Models: A Lie Algebraic View]() (Link coming soon)

## Instruction

The main directory is `state_tracking`.

* Please install [fla-org/flash-linear-attention](https://github.com/fla-org/flash-linear-attention) following the instructions provided in the link.
  All experiments were conducted using commit `8d25cac1e1779dad0f1e36cb033c5a9b10bf9c01` `(Date: Sat Dec 27 21:11:38 2025 +0800)`

* In order to install and use AUSSM models, we refer to the instructions provided at [arjunkaruvally/AUSSM](https://github.com/arjunkaruvally/AUSSM).
Note that AUSSM requires to disable mix precision and the corresponding backward kernel assumes sequence lengths to be a multiple of 8 (or 16? didn't check the detail). The exact code we used for our experiments can be obtained by applying the patch shown in the file `diff_AUSSM.txt` (this patch should not be crucial, though). Note that installing AUSSM is optional, if your goal is to use other models---the rest of the code can be used without installing AUSSM.

* Further instructions are provided under `state_tracking`.
